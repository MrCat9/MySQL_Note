查看MySQL安装路径：

1.打开navicat

2.新建查询

3.
select @@basedir




连接MySQL：

cd 到 MySQL 的 bin 目录下

λ mysql -u root -p




查看databases：

mysql> show databases;




选择database：

mysql> use (databaseName)




查看database下的表：

mysql> show tables;




创建数据库：

mysql> CREATE DATABASE [数据库名];

(创建表)

(mysql> alter table [表名] convert to character set utf8mb4;)  # 更改表的字符集

(mysql> SHOW CREATE TABLE [表名];)  # 查看表的字符集




删除数据库

drop database [数据库名];




清空表

truncate table [表名];
